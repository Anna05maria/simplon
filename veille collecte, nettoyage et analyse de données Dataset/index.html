COLLECTE, NETTOYAGE ET ANALYSE DE DONNÉES

LA COLLECTE : COMMENT ÇA MARCHE ?

La collecte consiste à rassembler des données brutes à partir de différentes sources.

Création de données

  • Vous créez un questionnaire (Google Forms, SurveyMonkey)
  • Les répondants saisissent leurs réponses
  • Le logiciel génère automatiquement un fichier (CSV, Excel)

Extraction de données web

  • Un outil comme Thunderbit scanne des pages internet
  • L'IA identifie automatiquement les informations utiles (prix, noms, emails)
  • Les données se synchronisent automatiquement quand vous êtes connecté

NETTOYAGE

Le nettoyage transforme des données brutes désordonnées en données propres et exploitables.

Profilage
  • Le logiciel analyse vos données et identifie les problèmes (valeurs vides, formats bizarres, doublons possibles)
Normalisation
  • Mise au même format de toutes les dates
  • Mise en minuscules de tout le texte
  • Conversion des unités (mètres/pieds, euros/dollars)
Correction des erreurs
  • Détection des fautes de frappe ("Dupont" vs "Dupond")
  • Correction automatique selon des règles que vous définissez
Gestion des valeurs manquantes
  • Soit suppression des lignes incomplètes
  • Soit remplacement par une valeur (moyenne, médiane, etc.)
Déduplication
  • Recherche des doublons même avec des variations ("Jean Dupont" vs "J. Dupont")
  • Fusion ou suppression des doublons identifiés
outils utiliser
  • Excel / Google Sheets(Nettoyage simple)
  • OpenRefine (Nettoyage intermédiaire)
  • Python avec Pandas (Nettoyage avancé)
  • Data Ladder (Nettoyage pro)
  • Winpure (Déduplication)
  • Melissa Clean Suite (Nettoyage en temps réel)

L'ANALYSE : COMMENT ÇA MARCHE ?

L'analyse consiste à explorer les données propres pour en tirer des informations et des conclusions.

Analyse descriptive
  • Calcul de moyennes, pourcentages, totaux
  • Création de graphiques (camemberts, barres, courbes)
  • Tableaux croisés
Analyse diagnostique
  • Recherche de corrélations entre variables
  • Comparaison de groupes
  • Tests statistiques
Analyse prédictive
  • Création de modèles à partir des données existantes
  • Prédiction pour de nouvelles données
outils utiliser
  • Excel / Google Sheets(Analyse simple)
  • Tableau (Visualisation interactive)
  • Power BI (Business intelligence)
  • Python (Analyse avancée)
  • R (Statistiques)
  • SPSS (Sciences sociales)
  • KNIME (Data mining)
  • RapidMiner (Machine learning)
  • QuestionPro (Analyse d'enquêtes)
